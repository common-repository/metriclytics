== Description ==

This plugin allows you to add MetricLytics code in order to monitor the actions of your users within your site, so as to generate marketin campaigns and advertising strategies.

== Installation ==

1. Upload `metriclytics` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Setting, search a MetricLytics option and insert your Web Token ID
